# Gaze tracking software/libraries

## Pigaze

https://github.com/CR1502/PiGaze

* designed for the raspberry pi
	* created for raspberry pi 3/4, might not be compatible with pi 5 (unlikely)
	* lists pi camera module as a requirement, may not work with usb webcam

## Pygaze
https://www.pygaze.org/

This is an overall eye tracking software, it seems to be aimed at academic/psyhology solutions. It might still be useful. 

* may be too powerful for raspberry pi
* it seems a lot more complex than other solutions I could find. This means it has more potential applications but it may also be too cumbersome to learn in our timeframe

## Gaze Tracking
https://github.com/antoinelame/GazeTracking

Python library with simple, webcam-based eye tracking. Detects left, right, and center gazes, and blinking. This seems to be closest to what we're trying to do but might not work on pi

## Gaze follower
https://github.com/GanchengZhu/GazeFollower

Another Python library, seems to be actively developed, unlike others in this list. Does not support blink detection currently, and seems to be designed for academic applications e.g. psychology and testing. May be too powerful for raspberry pi.
